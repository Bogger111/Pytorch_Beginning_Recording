# Pytorch_Beginning_Recording

## 项目简介

本仓库记录了我学习 PyTorch 和深度学习基础过程中的代码、笔记和小实验。

内容从 PyTorch 基础操作开始，逐步涉及自动求导、神经网络搭建、CNN、Batch Normalization、ResNet 和基础图像分类任务。

## 学习内容

- PyTorch 张量操作
- 自动求导机制
- 线性回归与基础神经网络
- 多层感知机
- 卷积神经网络 CNN
- MNIST 图像分类
- CIFAR-10 图像分类
- Batch Normalization
- ResNet 与 ResNet18
- 简单可视化与模型评估

## 仓库目的

这个仓库主要用于记录我的深度学习入门过程。相比只看教程，我更希望通过自己实现代码、调试模型和观察实验结果来理解深度学习的基本方法。

## 技术栈

- Python
- PyTorch
- Torchvision
- NumPy
- Matplotlib
